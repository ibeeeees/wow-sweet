# Wolf of Wall Sweet <img src="public/favicon.ico" alt="" width="40" height="40" style="vertical-align:middle;" />

**A Candy-Themed AI Agent Stock Market Simulation with Autonomous Trading Intelligence**

## Overview

500 publicly traded companies rendered as candy-themed storefronts in a zoomable 3D isometric city. **500,000 autonomous AI agents** powered by Gemini API race to candy stores representing tickers that will generate the most profit. Agents physically collide, bump, and compete in real-time based on live market data.

## Tech Stack

**Frontend:** React + TypeScript + Three.js + Vite + WebGPU

**Backend:** Databricks + Google Colab + Kaggle Stock Data

**AI:** Gemini API for agent intelligence

**APIs:** Alpaca (live trading), Webull API
